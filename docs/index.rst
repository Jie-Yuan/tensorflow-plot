TensorFlow Plot
===============

.. image:: https://img.shields.io/github/stars/wookayin/tensorflow-plot.svg?style=social
    :target: https://github.com/wookayin/tensorflow-plot
    :alt: Github

:mod:`tfplot` is a simple TensorFlow_ utility for drawing any matplotlib_ plots or figures into images, as a part of TensorFlow computation graph.
Especially, we can easily any plot and see the result image as an image summary in TensorBoard.
